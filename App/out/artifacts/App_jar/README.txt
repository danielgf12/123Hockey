============================
123Hockey - Versión Demo
============================

▶ PASOS PARA INICIAR LA APLICACIÓN

1. Abre XAMPP y asegúrate de que el servicio de MySQL esté iniciado.

2. Entra en phpMyAdmin o usa el terminal para importar la base de datos:
   - Si quieres usar la aplicación SIN datos: importa el archivo `123hockey.sql`
   - Si quieres usar la aplicación CON datos de prueba: importa el archivo `123hockey_DATOS.sql`

3. Ejecuta la aplicación:
   - Si importaste la base de datos CON datos: haz doble clic en `iniciarConDatos.vbs`
   - Si importaste la base de datos VACÍA: haz doble clic en `iniciarSinDatos.vbs`

⛔ IMPORTANTE:
- No cambies el nombre de los archivos `.jar` ni elimines la carpeta `lib/`.
- Asegúrate de que el puerto de MySQL es el 3306 (el predeterminado de XAMPP).
